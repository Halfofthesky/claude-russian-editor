# Сокращения в русском тексте

Правила использования сокращений в русском тексте. Источник: Мильчин А. Э., Чельцова Л. К. «Справочник издателя и автора» (глава 4).

## Пять видов сокращений

1. **Графические** — с точкой, дефисом или косой чертой: *рус.*, *пр-во*, *п/п*
2. **Инициальные аббревиатуры** — из первых букв: *СНГ*, *ИТАР*, *ЦСКА*
3. **Сложносокращённые слова** — усечённые части слов: *колхоз*, *авиазавод*
4. **Высекаемые слова** — с выброшенной серединой: *млрд*, *млн*
5. **Смешанные** — комбинация способов: *НИИполиграфмаш*

## Главные требования

Сокращение допустимо, если оно:
- **Соответствует тексту** — в художественной литературе сокращения нежелательны, в научной и справочной — целесообразны
- **Понятно читателю** — общепринятые без расшифровки; специальные — с расшифровкой при первом упоминании
- **Не создаёт омонимии** — *ЗИЛ* не должно читаться двояко в одном тексте
- **Экономит место** — сокращение из 4+ букв экономит мало; оправдано при частом повторении
- **Единообразно** — одно слово сокращается одинаково во всём издании

## Правила графических сокращений

### Точка при усечении
- Ставится при отсечении конца слова: *рус.*, *проф.*, *напр.*
- Не ставится у высекаемых слов: *млн*, *млрд*, *кг*, *м*
- Не ставится, если осталась последняя буква слова: *г-н* (господин), *г-жа*

### Дефис при высекании середины
- *пр-во* (производство), *б-ка* (библиотека), *ин-т* (институт)

### Косая черта
- *Ростов н/Д* (Ростов-на-Дону), *п/п* (по порядку), *х/б* (хлопчатобумажный)

### Множественное число
- Удвоением буквы: *вв.* (века), *гг.* (годы), *пп.* (пункты), *тт.* (тома)
- Исключение: *с.* (страницы — не удваивается)

### Падежные окончания
- У графических сокращений падежные окончания не ставятся: *в 5 т.* (не *в 5 т-х*)

## Правила для аббревиатур

### Написание
- Буквенные — заглавными без точек: *РФ*, *МГУ*, *ООН*
- Звуковые, ставшие нарицательными — строчными: *вуз*, *загс*, *нэп*
- С прописной, если обозначают собственное имя: *ООН*, *НАТО*

### Склонение
- Звуковые мужского рода склоняются: *в вузе*, *из ЗАГСа*
- Буквенные — не склоняются: *из МГУ* (не *из МГУ-а*)

### Род
- Определяется по главному слову: *ООН приняла* (организация — ж.р.)
- Звуковые могут приобретать грамматический род по форме: *ТАСС уполномочен заявить* (м.р. по звучанию, хотя «агентство» — ср.р.)

## Общепринятые сокращения (можно использовать без расшифровки)

### При именах и обращениях
*г-н* (господин), *г-жа* (госпожа), *т.* (товарищ), *им.* (имени)

### При географических названиях
*г.* (город), *с.* (село), *д.* (деревня), *обл.* (область), *р-н* (район)

### При датах и числах
*г.* (год), *гг.* (годы), *в.* (век), *вв.* (века), *до н. э.*, *н. э.*, *ок.* (около)

### При ссылках
*см.* (смотри), *ср.* (сравни), *напр.* (например), *т. е.* (то есть), *и т. д.* (и так далее), *и т. п.* (и тому подобное), *и др.* (и другие), *и пр.* (и прочие)

### Слова при числах
*тыс.* (тысяча), *млн* (миллион), *млрд* (миллиард), *экз.* (экземпляр), *руб.* (рубль), *коп.* (копейка)

## Чего нельзя делать

- Сокращать одно слово двумя способами в одном тексте
- Оставлять одну букву при сокращении: *с.* допустимо только для «страница» и «село»
- Сокращать до сочетания, совпадающего с другим: *ин.* — иностранный? институтский?
- Начинать предложение с сокращения в нижнем регистре
