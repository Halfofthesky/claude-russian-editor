# Перечни в русском языке: Справочные правила

## 1. Виды перечней

### Внутриабзацные перечни
Элементы размещены внутри абзаца, разделены запятыми или точками с запятой.

**Когда использовать:**
- Элементы состоят из одного или нескольких слов либо простых нераспространённых предложений
- Не требуется специального выделения каждого элемента
- Издание требует компактности

**Пример:**
> Выделяют светофильтры: 1) монохроматические, 2) селективные, 3) компенсационные, 4) субтрактивные.

### Перечни с элементами-абзацами
Каждый элемент размещён на отдельной строке и может занимать несколько строк.

**Когда использовать:**
- Элементы состоят из многих слов или нескольких предложений
- Требуется подчеркнуть и выделить каждый элемент
- Нужно облегчить поиск и запоминание информации

## 2. Обозначения элементов внутриабзацного перечня

### Арабские цифры с закрывающей скобкой (1), 2), 3))

**Рекомендовано при:**
- Абзац предваряется буквой (начинающей букву)
- Необходимы ссылки на элементы (цифровые ссылки удобнее)
- Много элементов (цифры компактнее букв)

**Пример:**
> По характеру поглощения света светофильтры делятся на следующие группы: 1) монохроматические, 2) селективные, 3) компенсационные, 4) субтрактивные.

### Строчные буквы с закрывающей скобкой (а), б), в))

**Рекомендовано при:**
- Абзац нумерован
- В тексте много цифр (буквы более наглядны)
- В рубрикации широко используются цифры

**Пример:**
> Не выделяются кавычками: а) названия сортов растений в специальной литературе; б) общепринятые названия цветов и плодов.

### Знаки (точки, кружки, ромбики, квадраты, звёздочки)

**Рекомендовано при:**
- Перечень не требует отсылок к элементам в определённом порядке
- Уже использованы цифры и буквы для элементов других уровней
- Нужно выделить примечания среди текста, документы, цитаты, примеры

**Пример:**
- Примечание 1 в тексте
- Пример вычисления
- Документ в приложении

## 3. Обозначения элементов перечня с абзацами-элементами

### Арабские цифры с закрывающей скобкой (1), 2))

**Структура:**
```
1) первый абзац-элемент перечня;
2) второй абзац-элемент перечня.
```

**Используется для:** Относительно небольших абзацев из одного нераспространённого предложения. Номера, литеры и родовые обозначения ставят перед рубриками для: 1) резкого разграничения рубрик разных ступеней; 2) упрощения ссылок на рубрики.

### Римские и арабские цифры с точкой

**Структура для многоступенного перечня:**
```
I. Первый элемент старшего перечня:
   1. Первый элемент подчинённого перечня;
   2. Второй элемент подчинённого перечня.
II. Второй элемент старшего перечня.
```

**Используется для:** Абзацы состоят из нескольких предложений, разделённых точкой.

### Строчные буквы с закрывающей скобкой (а), б), в))

**Структура:**
```
1. Первый элемент старшего перечня:
   а) подчинённый элемент;
   б) то же.
2. Второй элемент старшего перечня.
```

**Используется для:** Абзацы-элементы подчинены пронумерованным старшим абзацам. Подчинённые элементы просты и состоят из одной фразы либо являются однострочными перечнями (большие по объёму).

### Прописные буквы с точкой (А., Б., В.)

**Структура:**
```
I. Текст:

A. Текст:

1. Текст;
   а) текст;
   б) текст;

2. Текст.
```

**Используется для:** Перечень содержит элементы нескольких ступеней соподчинённости в порядке: I. А. 1. а).

### Знак тире (—)

**Структура:**
```
I. Первый элемент перечня.

A. Текст:

1. Текст:
   — текст;
   — текст.

   б) текст;
   и т.д.
```

**Используется для:** Перечень не требует запоминания элементов в определённом порядке. Элементы относятся к самой низшей ступени.

## 4. Знаки препинания

### Перед перечнем

**Двоеточие (:) ставится:**
- Если текст содержит слово/словосочетание, указывающее на перечень далее
- Перечень служит разъяснением предшествующего текста
- Перечень непосредственно продолжает текст

**Пример:**
> Виды светофильтров: 1) монохроматические; 2) селективные; 3) компенсационные.

**Точка (.) ставится:**
- Связь между текстом и перечнем ослаблена
- Нет указателя на перечень в предшествующем тексте
- Перечень не служит разъяснением или продолжением

### Между элементами внутриабзацного перечня

**Запятая (,):**
- Элементы простые (несколько слов, без внутренних знаков препинания)

**Точка с запятой (;):**
- Элементы содержат много слов
- Элементы имеют знаки препинания внутри

**Пример с запятыми:**
> Различаются шмуцтитулы, шапки и текст.

**Пример с точками с запятой:**
> Рекомендуется: цифры-номера с точкой; прописные буквы с точкой; номера-цифры с закрывающей скобкой.

### После абзацев-элементов перечня (кроме последнего)

**Запятая (,):**
- Абзацы очень просты (несколько слов, без внутренних знаков препинания)
- Обозначены номером-цифрой/буквой с закрывающей скобкой или наборным знаком (тире)
- Начинаются со строчной буквы

**Точка с запятой (;):**
- Элементы не совсем просты (имеют знаки препинания внутри)
- Обозначены номером-цифрой/буквой с закрывающей скобкой или наборным знаком
- Начинаются со строчной буквы

**Точка (.):**
- Элементы обозначены номером-цифрой/прописной буквой с точкой
- Начинаются с прописной буквы (не потому, что имя собственное)

**Правило согласованности:**
Между элементами ставится знак, требуемый для наиболее сложного элемента. Если один элемент требует точку с запятой, точка с запятой ставится после всех элементов.

### При примечании к элементу перечня

- После элемента перечня — **точка (.)**
- После примечания — знак препинания, принятый для межэлементного (например, **точка с запятой (;)**)

## 5. Прописные буквы в перечнях

### Когда начинать с прописной буквы

**Абзацы-элементы начинаются с прописной буквы, если:**

1. Текст элемента делится на предложения, разделённые точкой

2. Элемент обозначен номером-цифрой или прописной буквой с точкой

3. Элемент обозначен номером-цифрой или строчной буквой с закрывающей скобкой, но:
   - Содержит много слов
   - Сложен по структуре
   - Способ обозначения цифрой/буквой с точкой уже использован для элементов старшего перечня

**Примеры:**

Вариант 1 (цифры с точкой — начинать с прописной):
```
I. Первый элемент. Состоит из нескольких предложений.
II. Второй элемент. Здесь также прописная буква.
```

Вариант 2 (буквы с скобкой — строчная):
```
а) простой элемент;
б) ещё один простой элемент.
```

Вариант 3 (буквы с скобкой, но сложный — прописная):
```
а) Длинный и сложный элемент, содержащий много слов и несколько предложений.
б) Ещё один длинный элемент.
```

## 6. Принцип единообразия

**Общее правило:**
Однородные по значимости перечни оформляются одинаково, независимо от объёма и сложности структуры. Требуется осмысленное решение, основанное на анализе всех перечней в издании.

**Для нормативно-инструктивной литературы:**
- Подпункты 1-й ступени: строчные буквы с закрывающей скобкой (а), б), в))
- Подпункты 2-й ступени: знак тире (—)
- При необходимости 3-й ступени: используется согласованная с издателем схема

## 7. Текст перед перечнем

### Рекомендуемая форма
Перед перечнем ставят обобщающее слово (существительное, глагол, прилагательное в зависимости от ведущего слова элементов).

**Пример:**
> Выделяют следующие виды светофильтров: 1) монохроматические; 2) селективные.

### Допустимая форма
Если обобщающее слово невозможно ввести без дополнительных слов, можно завершить текст глаголом или расщепленным сказуемым, перенеся предлог в состав первого элемента.

**Пример:**
> Светофильтры делятся на: 1) монохроматические; 2) селективные.

### Нерекомендуемая форма (висячий предлог)
Не оставляйте предлог в предваряющем перечень тексте.

**Неправильно:**
> Светофильтры состоят из:
> 1) монохроматических;
> 2) селективных.

**Правильно:**
> Светофильтры состоят из следующих видов:
> 1) монохроматические;
> 2) селективные.
