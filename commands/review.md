Проведи комплексную редакционную проверку оформления русского текста.

Сначала прочитай навык `russian-editorial-review` из `${CLAUDE_PLUGIN_ROOT}/skills/russian-editorial-review/SKILL.md` — он содержит полный порядок проверки и маршрутизацию к специализированным навыкам.

Если пользователь указал файл ($ARGUMENTS), прочитай его. Если нет — попроси указать файл или вставить текст.

Выполни проверку в порядке, описанном в навыке:
1. Составь паспорт оформления
2. Проведи последовательную проверку по всем областям
3. Выполни вертикальный контроль
4. Сформируй отчёт с замечаниями, сгруппированными по областям

При необходимости обращайся к подробным правилам в `references/` каждого навыка.
